# Commit Guard Support

Public support and technical documentation for Commit Guard.

This repository hosts:

- the public GitHub Pages site for support and technical documentation
- the public issue tracker used for bugs and feature requests

Recommended public URLs:

- Home: `https://shopycornltd.github.io/commit-guard-support/`
- Docs: `https://shopycornltd.github.io/commit-guard-support/docs/`
- Issue tracker: `https://github.com/shopycornltd/commit-guard-support/issues`

## Site Structure

- `index.html` - support home page
- `docs/` - technical documentation
- `issue-tracker/` - public issue tracker landing page
- `assets/` - shared branding and styles
- `.github/ISSUE_TEMPLATE/` - public issue templates
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow

## Publishing

1. Push to `main`
2. In GitHub, set `Settings -> Pages -> Source = GitHub Actions`
3. The `Deploy GitHub Pages` workflow will publish the static site

## Notes

- Security issues should not be filed publicly. The site links users to the separate security reporting instructions.
- Legal pages stay on the main public Commit Guard website.
